# ALX Listing App

A modern Airbnb clone listing page built with Next.js, TypeScript, TailwindCSS, and ESLint.

## Project Structure

- **components/**: Reusable UI components (Card, Button)
- **interfaces/**: TypeScript interfaces for type safety
- **constants/**: Reusable constants and configurations
- **public/assets/**: Images and SVGs used in the project
